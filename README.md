#Video Object Detection using streamlit
# Prerequisites
'''
Install python and libraries mentioned below to execute code correctly 
1. Streamlit
2. Transformers Library Hugging Face
3. OpenCV
4. Pillow
5. Pytorch
'''

# Steps to Reproduce
'''
1 Download DETR model from huggingface and place it in your local directory and store path of local directory in model_path variable
2 Execute command streamlit run streamlit_video_detection_local.py
'''
